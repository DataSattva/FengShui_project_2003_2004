Меню «Файл» >> Персональный гороскоп

(кнопка на главном окне программы)

Команда вызывает окно персонального гороскопа. Расчет персонального гороскопа производится на основании пола, даты и времени рождения.
При первом вызове окна, устанавливаются текущие значения даты и времени и значения GMT, л.в. (летнее время) и долготы из общих установок программы.

Поля окна «Персональный гороскоп»

Поле «Имя» служит для ввода имени или другого идентификатора человека, на которого делается расчет.

Поле «Дата рождения» служит для ввода даты и времени рождения

Кнопки выбора пола служат для выбора пола, -- «ж» -- женский, «м» -- мужской.

Поле «GMT» служит для введения значения поясного времени. Для Москвы это составляет +3 часа относительно Гривничского времени.

Поле «л.в.» служит для введения значения летнего времени, которое было принято в месте и в момент рождения человека.

Поле «долгота». Если маленький квадрат перед словом «долгота» отмечен галочкой, то производится более точный расчет часа и дня рождения, исходя из значений долготы местности. Для Москвы это 37 градусов.

Поле «Отображать в календаре». Если маленький квадрат перед надписью «Отображать в календаре» отмечен галочкой, то при использовании команда «Расчет» происходит расчет на дату и время рождения не только персонального гороскопа, но и данных календаря и круга эклиптики.


Активные элементы окна «Персональный гороскоп»

После произведенного расчета, кроме всего прочего, в окне «Персональный гороскоп» появляются активные элементы -- подчеркнутые строки синего цвета. При нажатии на них левой кнопкой мыши, открываются окна с дополнительной информацией.

Ствол дня рождения (элемент ствола) -- открывается окно «Стволы и ветви» с установленным стволом дня рождения и установленной ветвью текущего месяца.

Характеристика звезды -- открывается окно с описанием характеристики звезды года рождения.

Система Ба чжай: восточная/западная судьба -- открывается окно «Ба чжай» с установленным в нем рассчитанным значением.
