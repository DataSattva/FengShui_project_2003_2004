Поля установки значений даты и времени для расчета календаря, круга эклиптики и наступления сезонов. Минимальное значение даты, которую можно установить в соответствующем поле 1 января 1900 года, максимальное значение -- 1 января 9000 года.

Кнопка восстановления текущих даты и времени в полях установки даты и времени основного окна программы.

Здесь задается значение числа дней шага для поля установки даты основного окна программы. Минимальное значение шага -- 0 дней, максимальное -- 99999 дней. Кнопки «+» и «-» используется для прибавления или вычитания значения шага к/от значения в поле даты основного окна программы соответственно. При активизации окна поля с числом дней кнопкам «+» и «-» соответствуют клавиши «Стрелка вверх» и «Стрелка вниз» на клавиатуре.

Если это поле не отмечено галочкой, то все изменения в полях установки даты и времени основного окна программы не будут отображаться в ее интерфейсных элементах. Это функция применяется для медленных компьютерах, когда требуется быстрая смена дат, без расчета всего календаря.
